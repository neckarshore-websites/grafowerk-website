# AGENTS.md

This repo is the Grafowerk marketing website (Next.js 16 / App Router / Tailwind
v4 / TypeScript). See `CLAUDE.md` for conventions.

Quick reference:

- Install: `npm install` · Dev: `npm run dev` · Build: `npm run build`
- Lint: `npm run lint` · Smoke tests: `npm run test:smoke:local`
- Dependencies are pinned to exact versions — keep it that way (`--save-exact`).
- Do not promote to production without explicit approval; push to `main` →
  Vercel preview.
